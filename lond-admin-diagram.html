📊 Status Codes ทั้งหมด

Statusภาษาไทยแผนกขั้นตอน
newเคสใหม่ฝ่ายขายเริ่มต้น
contactingกำลังติดต่อฝ่ายขายติดตาม
awaiting_appraisal_feeรอค่าประเมินฝ่ายขายรอชำระ
appraisal_scheduledนัดประเมินฝ่ายประเมินนัดหมาย
appraisal_in_progressกำลังประเมินฝ่ายประเมินดำเนินการ
appraisal_passedประเมินผ่านฝ่ายประเมิน✅ ผ่าน
appraisal_not_passedประเมินไม่ผ่านฝ่ายประเมิน❌ ไม่ผ่าน
pending_approveรออนุมัติฝ่ายอนุมัติรอพิจารณา
credit_approvedอนุมัติวงเงินฝ่ายอนุมัติ✅ อนุมัติ
rejectedไม่อนุมัติฝ่ายอนุมัติ❌ ปฏิเสธ
pending_auctionรอประมูลฝ่ายประมูลรอจับคู่นายทุน
auction_completedโอนทรัพย์เสร็จสิ้นฝ่ายประมูล✅ โอนแล้ว
preparing_docsเตรียมเอกสารฝ่ายออกสัญญาร่างสัญญา
legal_scheduledนัดนิติกรรมฝ่ายนิติกรรมนัดวัน

👤 Sales Data Isolation

ฝ่ายขาย (department=sales) จะเห็นเฉพาะข้อมูลของตัวเอง:

• ID ลูกหนี้ที่ตัวเองสร้าง → loan_requests.created_by_id = user.id

• ID เคสที่ถูก assign → cases.assigned_sales_id = user.id

super_admin เห็นทุกอย่าง (ไม่มี WHERE filter)
